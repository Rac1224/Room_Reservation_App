# Library Room Reservation Application

This project aims to keep track of the student room reservtaion records and allows users like students and librarians to reserve or mordify rooms in the library for group discussion, study sesion or recession.

## Software tools
- Microsoft Visio
- Visual Studio
- Abode Photoshop
- .NET

## Programming Languages
- SQL
- C#
